# Healthcare-Analytics-Dashboard
Healthcare Analytics Dashboard using SQL, Python and Power BI

## Project Overview

This project analyzes healthcare data to understand patient behavior, appointment trends, treatment performance, billing status, and hospital revenue.

## Tools Used

- SQL
- Python
- Pandas
- Matplotlib
- Seaborn
- Power BI

## Project Workflow

1. Data Cleaning
2. SQL Analysis
3. Exploratory Data Analysis (EDA)
4. Business Insights
5. Interactive Dashboard Development

## Dashboard Pages

### Executive Summary
- Total Patients
- Total Revenue
- Appointment Trends
- Revenue by Treatment

### Patient & Doctor Analysis
- Specialization Demand
- Doctor Performance
- Patient Demographics
- Visit Reasons

### Financial Performance
- Revenue Analysis
- Payment Status
- Payment Methods
- Treatment Revenue Share

## Key Findings

- Pediatrics recorded the highest appointment volume.
- Chemotherapy generated the highest revenue.
- Credit Card was the preferred payment method.
- Pending transactions exceeded paid transactions.
- MRI and X-Ray contributed significantly to revenue.
